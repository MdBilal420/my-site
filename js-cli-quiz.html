A Quiz app on javascript CLI

November 25, 2020
In this blog, we will learn how to create a Command Line Application on repl.it . We will be creating a quiz app on a famous tv show ‘SUITS’.

Two dependancies that we will be using are:

  1. readlineSync : readlineSync tries to let your script have a conversation with the user via a console.
  2. chalk: chalk is used for terminal string styling.

var readlineSync = require('readline-sync');

const chalk = require('chalk');

Using above line of code, repl.it will automatically add those dependancies on to package.json file.

"dependencies": {"chalk": "^4.1.0","readline-sync": "^1.4.10"}

Now, we can begin to write our quiz app.

Take the Player name as input.

var username = readlineSync.question('Enter Your Name: ');

Instructions

console.log(chalk.red('Suits Quiz'));

console.log(chalk.green('A quiz from our favourite tv show'));

console.log(chalk.yellow('Write the correct answer.'));

console.log('Points: Correct: +3 Incorrect: -1');

Add Questions to the javascript object.

var questions = [

{question: `Who is the female shark in charge of the 'Suits' law firm?`, A: 'Rachel Zane', B: 'Jessica Pearson', C:'Donna Paulsen', answer: 'b'},

{question: `Who lands an interview with Harvey Specter after he runs from a drug deal gone bad?`, A: 'Jeff Malone', B: 'Sean Cahill', C:'Mike Ross', answer: 'c'},

{question: `The firm hires law graduates from ________?`,A: 'Harvard', B:'Stanford', C:'Oxford', answer: 'a'},

{question: `How does Louis figure out that Mike didn't attend Harvard?`, A: 'Knight of Columbus', B: 'Key Largo', C:'Order of the Coif key', answer: 'c'},

{question: `What law exam did Mike charge to take for others?`, A: 'SAT', B: 'LSAT', C:'ACT', answer: 'b'},

];

Here, we are giving 3 options for each question. We can take any questions according to our preference and provide options to A,B and C respectively.

We will make use of two variables.

var score = 0;

var best = 5;

Now, we need to log these questions and options on console. For that we will make use of a for loop.

Print Questions and Options on screen and take input as answer

var i = 0;i<questions.length;i++)

{

var n = i+1;

//question

console.log(chalk.inverse(n+'. '+questions[i].question));

//options

console.log(chalk.cyan('A: '+questions[i].A));

console.log(chalk.cyan('B: '+questions[i].B));

console.log(chalk.cyan('C: '+questions[i].C));

// Input Answer

var answer = readlineSync.question('Answer: ');

//compare answers

if(answer.toLowerCase() === questions[i].answer.toLowerCase()){//on each correct answer print 'correct' and add 3 to 'score'

console.log(chalk.bgGreen('Correct'));

score = score + 3;}

//on incorrect answer print 'incorrect' and subtract -1

else{

console.log(chalk.bgRed('Incorrect'));

score = score -1; }

Compare if current score is greater than best score.

if(score > best){

//congratulate if score is greater than best

console.log(chalk.yellow('Congratulations ! This is the best score \n')

// update best score

best = score

}

Print final Score

console.log(chalk.red(username +' your final score is: '+ score));

Now, our app is complete.
